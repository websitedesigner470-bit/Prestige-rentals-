# Prestige Rentals | Luxury Without Limits

Australia's premier concierge service for supercars, superyachts, and private aviation.

## Tech Stack
- Framework: Next.js 15
- Styling: Tailwind CSS
- Animation: Framer Motion
- Deployment: GitHub Pages

## Setup Instructions
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for deployment: `npm run build`

## Brand Assets
- Primary Colors: Black (#050505), Charcoal (#111111)
- Typography: Playfair Display (Headings), Inter (Body)
